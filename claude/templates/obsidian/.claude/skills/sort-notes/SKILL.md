## 指示

指定された日付・期間のデイリーノートを読み取り、内容をトピックごとに分類して `Categories/` 配下の分類別ファイルに追記してください。

対象: $ARGUMENTS（例: `2026-02-07`, `2026-02-01 2026-02-07`, 未指定の場合は当日）

### 手順

1. 対象のデイリーノート（`Daily/<日付>.md`）を読み取る
2. ノート内の各記述をトピック/カテゴリに分類する
   - プロジェクト名、作業種別、会議名など内容から判断
   - 既存の `Categories/` 配下のファイルがあれば、それに合わせて分類
3. 分類先ファイルに追記する:
   - ファイルパス: `Categories/<カテゴリ名>.md`
   - ファイルが存在しない場合は新規作成
   - 存在する場合は末尾に追記
4. 追記フォーマット:

```markdown
## YYYY-MM-DD

- 該当する内容をここに記載
- 複数項目がある場合は箇条書き
```

### ルール

- **元のデイリーノートは一切変更しない**（読み取り専用）
- 追記前に分類内容をユーザーに確認する（どのカテゴリに何を振り分けるか一覧表示）
- ユーザーが承認してから実際にファイルへ書き込む
- 同じ日付の内容が既に追記済みの場合は重複追記しない
- カテゴリ名は日本語でも英語でも可（既存ファイルがあればそれに合わせる）

## 指示

指定された日付・期間のデイリーノートを読み取り、セクションと `#ref/xxx` タグに基づいて分類別ファイルに追記してください。

対象: $ARGUMENTS（例: `2026-02-07`, `2026-02-01 2026-02-07`, 未指定の場合は当日）

### セクションと振り分け先の対応

| デイリーノートのセクション | 振り分け先 |
|---|---|
| `## work` 配下の `### #ref/xxx` | `work/xxx.md` |
| `## tech` 配下の `### #ref/xxx` | `tech/xxx.md` |
| `## private` 配下の `### #ref/xxx` | `private/xxx.md` |
| `## memo` | 振り分けなし（スキップ） |

### 手順

1. 対象のデイリーノート（`daily/<日付>.md`）を読み取る
2. 各セクション（`## work`, `## tech`, `## private`）内の `### #ref/xxx` ブロックを特定する
3. `xxx` の部分をファイル名として、対応するディレクトリ配下のファイルに追記する
4. 追記フォーマット:

```markdown
## YYYY-MM-DD

- 該当する内容をここに記載
- 複数項目がある場合は箇条書き
```

### ルール

- **元のデイリーノートは一切変更しない**（読み取り専用）
- 追記前に分類内容をユーザーに確認する（どのファイルに何を振り分けるか一覧表示）
- ユーザーが承認してから実際にファイルへ書き込む
- 同じ日付の内容が既に追記済みの場合は重複追記しない
- ファイルが存在しない場合は新規作成、存在する場合は末尾に追記

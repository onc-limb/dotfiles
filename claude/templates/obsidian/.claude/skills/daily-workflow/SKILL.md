## 指示

デイリーノートに対して以下の 3 つのスキルを順番に実行してください。

対象日付: $ARGUMENTS（未指定の場合は今日の日付を使用）

### 実行順序

1. `/refine-daily <日付>` — デイリーノートの不足情報を対話形式で補完
2. `/sort-notes <日付>` — デイリーノートの内容を分類別ファイルに振り分け
3. `/daily-report <日付>` — デイリーノートから日報を作成

### 手順

1. `$ARGUMENTS` が未指定の場合は `date +%Y-%m-%d` で当日の日付を取得する
2. Skill ツールを使って `refine-daily` を日付引数付きで呼び出す
3. `refine-daily` が完了したら、Skill ツールを使って `sort-notes` を同じ日付引数で呼び出す
4. `sort-notes` が完了したら、Skill ツールを使って `daily-report` を同じ日付引数で呼び出す
5. 全スキルが完了したら、実行結果のサマリーを表示する

### 注意事項

- 各スキルは必ず順番に実行する（前のスキルが完了してから次を実行する）
- 各スキルの対話的なやり取りはそのまま行う（ユーザーへの質問をスキップしない）
- いずれかのスキルでエラーが発生した場合、ユーザーに報告して続行するか確認する
- 各スキル間で「次のステップに進みます」と伝えてからスキルを呼び出す

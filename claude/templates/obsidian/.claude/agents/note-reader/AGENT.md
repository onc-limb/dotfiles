指定された期間のデイリーノートを全て読み取り、構造化された要約を返してください。

## 手順

1. `daily/` 配下から指定期間のノートファイルを特定する
2. 全てのノートを読み取る
3. 以下の形式で構造化された要約を返す:
   - 週ごとのグルーピング
   - 各日の主要な作業内容（箇条書き）
   - 期間全体を通じた主要トピック・プロジェクトの一覧
   - 定量情報（作業時間、タスク完了数等）があれば集計

## 出力形式

```
## 期間: YYYY-MM-DD 〜 YYYY-MM-DD

### 主要トピック
- トピック1
- トピック2

### 週次内訳

#### 第1週（MM/DD - MM/DD）
- MM/DD: 主な作業
- MM/DD: 主な作業
...

### 定量情報
- 総作業日数: N日
- （その他集計可能な数値）
```
